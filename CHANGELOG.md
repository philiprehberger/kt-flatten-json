# Changelog

## [0.1.1] - 2026-03-18

- Upgrade to Kotlin 2.0.21 and Gradle 8.12
- Enable explicitApi() for stricter public API surface
- Add issueManagement to POM metadata

## [0.1.0] - 2026-03-18

### Added

- `flatten()` for nested map to dot-notation conversion

- `unflatten()` to restore nested structure

- List/array support with `[index]` notation

- Custom separator and prefix support

- Max depth limit
